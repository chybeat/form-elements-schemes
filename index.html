About this project

This project use CSS language to make better and standarized (normalized) dark and light schemes across browsers on every HTML forms elements.

This page use prism.js to shows colored code like code editor and normalize v 8.0.1 to initialize styles in every browser. I recomend to use normalize or another CSS code to uniform rendered elements across browsers on all your projects.

The goal is colorize and make better default-project form elements (input, select, textarea, etc). The file form-schemes.min.css which stylize the form elements contains only a box model format for these specific elements and colors, is not configured with position areas or accomodates on parent elements with flex, grid or position rules. Just use before your own CSS or styles you want.

May is needed to clarify this project has some positioned elements, but is only for your undestanding, if you want to modify watch de unminified/unzipped version of files you can find in github folder to understand what changed in default box model, light scheme color or dark scheme color.
