Dark and light schemes for form elements

This project use CSS language to make better dark scheme for HTML forms elements, This page use prism.js to shows colored code like code editor and normalize v 8.0.1 to initial style to this page.

This project

The goal is colorize and make better default form elements (input, select, textarea, etc). The file forms-aio.min.css contains only a box model format for elements and colors, is not configured with position areas or accomodates on parent elements like flex, grid or position rules. Just use before your own CSS or styles you want.

May is needed to clarify this project has some positioned elements, but is only for your undestanding, if you want to modify, watch de unminified/unzipped version of files you can find here to undestand what is changed in box model,light scheme colors or dark scheme colors.

.dark-scheme and .light-scheme

				
The CSS detect via media query the dark or light scheme based on browser/OS current scheme to colorize
					the elements, but you can force a scheme using dark-scheme or
					light-scheme in parent container or element
