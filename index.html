Dark and light themes for form elements

Welcome, this repository shows the settings in css language to manage a dark theme in HTML forms elements, using prism.js to shows code and normalize v 8.0.1 to style setup to this page.

How to use

Download the file forms-aio.min.css and write link tag in head area of HTML file like below:


			

			

				
Other files

				
The css folder in repository can find and download next files:

				

					
						

							Filename
							Description
						

					
					
						

							formsBoxModel.css
							Box model CSS properties for form elements (Required for both schemes)
						

						

							darkFormElements.css
							Dark scheme (only colors) for form elements
						

						

							lightFormElements.css
							Light scheme (only colors) for form elements
						

						

							forms-aio.css
							(Recomended) Light and dark schemes along with box model rules for form elements
								in a single file
						

					
					
						

							All files includes the minified or compressed version and map to sources located in
								scss folder
